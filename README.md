This is a simple Android Weather App 

## Features
- Allows users to enter a city name
- Displays current weather conditions (simulated or static)
- Shows weather-related messages based on user input and conditional logic (e.g., sunny, rainy, cloudy)
- User-friendly interface with basic Android UI components

## Learning Objectives
- Practice user input handling using EditText and Button
- Implement `if` statements for decision-making
- Connect Kotlin code with XML layouts
- Build and test a native Android application

## Technologies Used
- Kotlin
- Android Studio
- XML (UI Layouts
